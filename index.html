Установка

Три шага

Выберите платформу — инструкции адаптируются автоматически. Нужен платный план: Claude Pro / Team или ChatGPT Plus / Team.

Отмечайте шаги по мере выполнения (Claude):

1
Скачайте файл скилла

Нажмите кнопку ниже — файл SKILL_marketing-strategy.md сохранится на компьютер. Это обычный текстовый файл с инструкциями для Claude.

💡
Файл весит ~42 КБ. Открыть можно в любом текстовом редакторе — Блокнот, TextEdit, VS Code. Нужен платный план Claude Pro или Team.
2
Создайте Проект в Claude.ai

Проекты — отдельные рабочие пространства в Claude. Скилл работает автоматически в каждом чате внутри проекта.

claude.ai
1. Откройте claude.ai → войдите в аккаунт 2. В левом меню найдите «Проекты» (Projects) 3. Нажмите «Создать проект» (Create project) 4. Назовите: «Маркетинговая стратегия» 5. Нажмите «Создать»
3
Вставьте текст файла в инструкции проекта

Откройте скачанный файл — выделите всё (Ctrl+A / Cmd+A) — скопируйте — вставьте в настройки проекта.

claude.ai → Настройки проекта
1. Зайдите в созданный проект 2. Нажмите «Project instructions» или ⚙ 3. В поле вставьте весь текст из скачанного файла 4. Нажмите «Save» 5. Создайте новый чат внутри проекта — готово
✓ Занимает 2 минуты Нужен Claude Pro или Team Повторять не нужно
1
Скачайте файл промпта

Нажмите кнопку ниже — файл PROMPT_marketing-strategy-gpt.md сохранится на компьютер. Внутри системный промпт для вставки в ChatGPT.

💡
Нужен план ChatGPT Plus или Team. На бесплатном тарифе Custom GPT недоступны.
2
Создайте Custom GPT

Custom GPT — это ваш персональный ассистент с заданным поведением. Работает как отдельная точка входа, без необходимости вставлять промпт каждый раз.

chatgpt.com
1. Откройте chatgpt.com → войдите в аккаунт 2. В левом меню нажмите на «Explore GPTs» 3. Нажмите «+ Create» (кнопка вверху справа) 4. Перейдите на вкладку «Configure» 5. Заполните поле Name: «Marketing & Consulting Strategy»
3
Вставьте промпт в поле Instructions

Откройте скачанный файл — выделите весь текст (Ctrl+A / Cmd+A) — скопируйте — вставьте в поле Instructions в Configure.

chatgpt.com → GPT Builder → Configure
1. В поле «Instructions» вставьте весь текст файла 2. В поле «Conversation starters» добавьте примеры: «Сделай маркетинговую стратегию» «Проведи аудит нашего маркетинга» 3. Нажмите «Save» → выберите «Only me» или «Anyone» 4. Откройте созданный GPT и начните чат — готово

FAQ

Частые вопросы

Работает ли на бесплатном плане Claude? ▼
Нет. Функция Projects (Проекты), через которую устанавливается скилл, доступна только на платных планах: Claude Pro ($20/мес) и Claude Team ($30/мес на человека). На бесплатном плане Projects недоступны.
Можно ли использовать скилл в нескольких проектах? ▼
Да. Создайте столько проектов, сколько нужно, и вставьте содержимое файла в каждый. Например, отдельный проект под каждого клиента или продукт. Файл можно редактировать под конкретный контекст.
Данные о компании попадают к Anthropic? ▼
Claude обрабатывает данные так же, как любой другой чат. На тарифе Claude Pro содержимое разговоров не используется для обучения моделей по умолчанию. На Claude Team действуют корпоративные условия конфиденциальности. Не передавайте в чат коммерческую тайну или персональные данные клиентов.
Скилл работает только для русскоязычного контента? ▼
Нет. Скилл написан на русском, но Claude понимает запросы на любом языке. Если написать по-узбекски или по-английски — он ответит на том же языке. Региональный блок содержит данные по ЦА и СНГ, но скилл работает и для других рынков.
Нужно ли обновлять скилл? ▼
Бенчмарки и данные внутри актуальны на 2024-2025 год. Структура скилла не устаревает. Если выйдет обновлённая версия — просто замените текст в Project instructions на новый. Удалять и пересоздавать проект не нужно.
Что делать, если документ получился слишком общим? ▼
Чем конкретнее вы ответили на 8 вопросов при старте — тем точнее результат. Если раздел кажется общим, используйте команду /update [номер раздела] и добавьте больше деталей: реальные цифры, конкретные конкуренты, примеры из практики.
Чем отличаются версии для Claude и ChatGPT? ▼
Содержание стратегии одинаковое — разделы, бенчмарки, фреймворки. Разница в формате файла: версия для Claude содержит YAML-заголовок и устанавливается через Projects. Версия для ChatGPT оформлена как чистый системный промпт для Custom GPT Builder. Команды-шорткаты (/update, /summary и другие) работают в обеих платформах.
Можно ли использовать промпт в ChatGPT без Plus? ▼
Да, с ограничением. Вставьте текст файла в начало чата как первое сообщение — напишите «Используй эти инструкции:» и вставьте содержимое. Промпт будет работать в рамках одного разговора, но при открытии нового чата его нужно вставлять снова. Custom GPT (требует Plus) решает эту проблему.
